Qu'est-ce que le positionnement ?

Le positionnement permet de modifier le cours classique de la mise en page pour produire des effets intéressants.
Il y a différents types de positionnement que vous pouvez appliquer à des éléments HTML. Pour utiliser un type particulier de positionnement sur un élément, nous utilisons la propriété position.

Positionnement statique

Le positionnement statique est celui reçu par défaut par chaque élément. Cela veut tout simplement dire « positionner l'élément selon le flux normal, rien de spécial à voir ici ».


Exemple:
p class=positioned


.positioned { position: static; background: yellow; }

Positionnement relatif

Le positionnement relatif est très similaire au positionnement statique. Cependant, une fois que l'élément positionné occupe une place dans le cours normal de la mise en page, vous pourrez modifier sa position finale. Vous pourrez par exemple le faire chevaucher d'autres éléments de la page.

Exemple:
position: relative; top: 30px; left: 30px;

Positionnement absolu

Le positionnement absolu nous apporte des résultats bien différents.
Si aucun élément ancêtre ne voit sa propriété position explicitement définie, par défaut, tous les éléments ancêtres auront une position statique et par conséquent, l'élément positionné de façon absolue sera contenu dans le bloc englobant initial.

Exemple:
position: absolute;

Positionnement fixe

fonctionne exactement de la même manière que le positionnement absolu, avec une différence essentielle : alors que le positionnement absolu fixe un élément en place par rapport à l'élément ou son parent positionné le plus proche, le positionnement fixe fige un élément en place par rapport à la vue par la fenêtre du navigateur elle-même. Cela signifie que vous pouvez créer des éléments d'interface utilisateur utiles qui sont fixés en place, comme des menus de navigation persistants.

Exemple:
body { width: 500px; height: 1400px; margin: 0 auto; }

Positionnement adhérent Sticky

Il existe une autre valeur de positionnement disponible : position: sticky. Elle est un peu plus récente que les autres. Il s'agit essentiellement d'un hybride entre position relative et position fixe : l'élément à positionner est en positionnement relatif jusqu'à un certain seuil (par exemple, 10px du haut de la fenêtre), seuil au-delà duquel il est en positionnement fixe.

Exemple:
.positioned { position: sticky; top: 30px; left: 30px; }

Qu'est-ce que la propriété Display ?

La propriété Display spécifie si/comment un élément est affiché.display
Chaque élément HTML a une valeur d’affichage par défaut en fonction du type d’élément dont il s’agit. La valeur d’affichage par défaut pour la plupart des éléments est ou .blockinline

Display: none

display: none; est couramment utilisé avec JavaScript pour masquer et afficher des éléments sans les supprimer ni les recréer. Jetez un oeil à notre dernier exemple sur cette page si vous voulez savoir comment cela peut être réalisé.
L’élément est utilisé par défaut:
"script display: none;"

Display: inline

Comme mentionné, chaque élément a une valeur d’affichage par défaut. Cependant, vous pouvez remplacer cela. Changer un élément en ligne en un élément de bloc, ou vice versa, peut être utile pour donner à la page une apparence spécifique, tout en suivant les normes Web. Un exemple courant est la création d’éléments en ligne pour les menus horizontaux :

Exemple:
li { display: inline; }

Display: block

La définition de la propriété display d’un élément ne modifie que la façon dont l’élément est affiché, PAS le type d’élément dont il s’agit. Ainsi, un élément en ligne avec n’est pas autorisé à avoir d’autres éléments de bloc à l’intérieur.display: block;


Exemple:
span { display: block; }

Display: inline-block

Par rapport à , la différence majeure est que permet de définir une largeur et une hauteur sur l’élément.display: inlinedisplay: inline-block De plus, avec , les marges / rembourrages supérieur et inférieur sont respectés, mais avec ils ne le sont pas. display: inline-blockdisplay: inline Par rapport à , la principale différence est qu’il n’ajoute pas de saut de ligne après l’élément, de sorte que l’élément peut rester à côté d’autres éléments.display: blockdisplay: inline-block L’exemple suivant illustre le comportement différent de , et :display: inlinedisplay: inline-blockdisplay: block

Exemple:
span.b { display: inline-block; width: 100px; height: 100px; padding: 5px; border: 1px solid blue; background-color: yellow; }

Display: flex

Comme nous l’avons spécifié dans le chapitre précédent, il s’agit d’un conteneur flexible (la zone bleue) avec trois éléments flexibles.
Le conteneur flex devient flexible en définissant la propriété sur :displayflex.

Exemple:
.flex-container { display: flex; }
